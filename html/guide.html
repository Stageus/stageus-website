Visual Studio Code 설치 방법

- 파이썬 응용 프로그램 과정과 웹 기초 과정 모든 팀원에게 필요합니다.
- 코딩 전용 문서 작업 프로그램으로, 줄여서 VSC라고 부릅니다.
- VSC를 통해 대부분의 프로그래밍 언어를 작성하고 실행할 수 있습니다.

(1) Visual Studio Code의 홈페이지에 접속한 뒤 우측 상단의 Download를 눌러주세요.

(2) 자신의 컴퓨터 운영체제 맞는 버전을 설치해줍니다.
맥북이 아닌 이상, Windows를 설치하면됩니다.

(3) 동의합니다에 체크 후 다음을 눌러주세요.

(4) 따로 선택할 것 없이 다음을 눌러주세요.

(5) 설치를 눌러주세요.

(6) 설치가 진행됩니다.

Anaconda Python 설치 방법

- 파이썬 응용 프로그램 과정 팀원에게 필요합니다.
- 기본적인 Python과 함께 유용한 모듈을 같이 포함하고있는 Python 버전입니다.
- 과정 중 배우는 SQLite3와 PyQt 등도 포함되어 있어 간편한 개발 환경 구성이 가능해집니다.

(1) Anaconda Python의 홈페이지에 접속한 뒤 우측 상단의 Get Started를 눌러주세요.

(2) Download Anaconda Installers를 눌러주세요.

(3) 자신의 컴퓨터 운영체제 맞는 버전을 설치해줍니다.
맥북이 아닌 이상, 64-Bit Windows를 설치하면됩니다.

(4) Next를 눌러주세요.

(5) I Agree를 눌러주세요.

(6) 따로 선택할 것 없이 Next를 눌러주세요.

(7) Next를 눌러주세요.
간혹 경로에 한글이 들어가면 설치가 되지 않는 경우가 있습니다.
이 경우 한글이 들어있지 않은 다른 경로로 변경해주세요.

(8) 꼭 첫 번째 체크박스를 체크하시고 Install을 눌러주세요.

(9) 설치가 진행됩니다.

(10) Next를 눌러주세요.

(11) 설치가 완료되었습니다.
